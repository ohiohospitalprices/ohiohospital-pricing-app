INTERACTIVE ELEMENTS:
================================================================================

1. HOSPITAL DROPDOWN
   Location: Top of controls section
   What it does: Filter results by single hospital
   Options: All Hospitals (default), Columbus, Cleveland, Dublin, Grady,
            Grant, Grove City, Hardin, Mansfield, Marion, O'Bleness,
            Pickerington, Riverside, Shelby, Southeastern, Van Wert
   Styling: Blue border on focus, smooth dropdown animation

2. CATEGORY BUTTONS (8 total)
   Location: Below hospital dropdown
   What they do: Filter results by procedure category
   Styling: 
     - Normal: White background, gray border
     - Hover: Purple text, border highlight
     - Active: Purple gradient background, white text
   Action: Click to select, click again to deselect (or click "All Categories")

3. SEARCH BOX
   Location: Below category buttons
   What it does: Real-time filtering by procedure name or CPT code
   Styling: Clean white box with purple focus border
   Action: Type to filter, clear to reset

4. PROCEDURE ITEMS
   Location: Results section
   What they show: Procedure name, price, CPT code, hospital name
   Styling: White box with subtle border, hover effect lifts box
   Action: Click to open comparison modal with all hospital prices

5. PRICE COMPARISON MODAL
   Location: Fullscreen overlay (centered)
   What it shows: All hospitals with pricing for selected procedure
   Styling: White popup on dark overlay, sorted by price
   Action: Click X or outside popup to close

USER WORKFLOW EXAMPLES:
================================================================================

EXAMPLE 1: Find all Surgical procedures at Columbus
────────────────────────────────────────────────────
1. Click Hospital Dropdown → Select "Columbus"
2. Click Category Button → Select "Surgical"
3. Results show: All surgical procedures at Columbus, sorted A-Z
4. Price shown: Click any procedure to see if available elsewhere

EXAMPLE 2: Search for a specific procedure
────────────────────────────────────────────
1. Click search box
2. Type "Appendectomy"
3. Results filter in real-time
4. Shows Appendectomy at all hospitals
5. Click to see price comparison across all locations

EXAMPLE 3: Compare prices for same procedure
──────────────────────────────────────────────
1. Reset filters (click "All Hospitals" and "All Categories")
2. Search for procedure (e.g., "CT Scan")
3. Click first result
4. Modal shows: All hospitals offering CT Scan with prices
5. Sorted from lowest to highest - find cheapest option!

EXAMPLE 4: Browse procedures by category
───────────────────────────────────────────
1. Select Hospital: "Cleveland"
2. Select Category: "Lab"
3. Browse all lab procedures at Cleveland
4. Click any to see if same procedure available at other hospitals
